This is Release Music Framework
RMF will manage you to actually finish your tracks in a few clear steps
  • Listen to your track
  • Write feedback on SoundCloud
  • Do at least 1 task per week
  • Render each working session and upload to SoundCloud
  • Are there any tasks left? Start new cycle
  • No? Release this track now!
  • Use my Mixer if any doubts of which version is better, compare different renders side-by-side and time-synced at simple online player
  • More details of my workflow in my recent Notion post
