This is Release Music Framework
RMF will manage you to really finish your tracks in a few clear steps
  • Listen to your track and write feedback on SoundCloud
  • Install this free Chrome extension to work with comments as they are tasks
  • Choose a task to start with today
  • Render each working session and upload to SoundCloud
  • Are there any tasks left? Do at least 1 per week
  • No? Release this track now!
  • Use this free Mixer if any doubts of which version is better, compare different renders side-by-side and time-synced at simple online player
  • Find more details about RMF workflow in the Notion post
